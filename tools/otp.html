One-Time-Pad

How it works:

  1. Input Text in Binärcode umwandeln
  2. Input Password in Binärcode umwandeln
  3. Länge des Passworts auf die Länge des Textes bringen
  4. Die beiden mit einander verrechnen
  5. Berechneten Binärcode wieder in Text umwandeln und ausgeben
